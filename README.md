➕ Addition Machine

A visually animated addition calculator built using Java Swing. This project simulates a retro-style addition machine with colorful buttons, flip-counter displays, smooth animations, and an interactive GUI.

🚀 Features
Interactive graphical user interface (GUI)
Enter two numbers and calculate their sum
Animated flip-style number display
Colorful retro-inspired button design
Custom painted calculator body using Java Graphics2D
Hover and click button effects
Clear (C) button to reset inputs
Beginner-friendly Java Swing project
🛠️ Built With
Java
Java Swing
AWT Graphics2D
Object-Oriented Programming (OOP)
